# 주식 관심종목 (Stock Watch)

한국 주식시장(KRX) 전용 인터랙티브 터미널 앱입니다. 종목명으로 검색해
관심종목을 등록하고, 실시간으로 시세를 감시할 수 있습니다.
데이터는 [FinanceDataReader](https://github.com/FinanceData/FinanceDataReader),
화면은 [rich](https://pypi.org/project/rich/) 로 구성했습니다.

## 특징

- **화면 안에서 관리** — 한 번 실행하면 메뉴에서 추가/수정/삭제 (별도 명령 불필요)
- **종목명 검색** — 정확한 코드를 몰라도 회사 이름(예: `삼성`, `카카오`)을 입력하면
  유사한 종목 후보가 추천되고 번호로 선택
- **종목명 표시** — 모든 화면에서 코드 대신 사람이 읽기 쉬운 종목명을 우선 표시
- **실시간 감시(watch)** — 자동 갱신되는 라이브 화면, 한국 시장 관례에 따라
  상승=빨강 / 하락=파랑
- **관련 뉴스** — 감시 모드 하단에 관심종목 관련 최신 뉴스 최대 5건 자동 표시
  (Naver Finance · Yahoo Finance · Bloomberg, 5분마다 갱신)
- 관심종목은 `data/watchlist.json` 에 저장되어 재실행해도 유지

## 설치

```cmd
pip install -r requirements.txt
```

Python 3.11+ 필요.

## 실행

```cmd
run.bat
```

또는:

```cmd
python app\main.py
```

실행하면 관심종목 표와 메뉴가 나타납니다:

```text

│ 주식 관심종목 │
╰───────────────╯
                          관심 종목
┏━━━┳━━━━━━━━━━┳━━━━━━━━┳━━━━━━━━━┳━━━━━━━━┳━━━━━━━━━┳━━━━━━━━━┓

┡━━━╇━━━━━━━━━━╇━━━━━━━━╇━━━━━━━━━╇━━━━━━━━╇━━━━━━━━━╇━━━━━━━━━┩

│ 2 │ 카카오   │ 035720 │  41,950 │ +4.61% │  44,000 │  40,450 │
└───┴──────────┴────────┴─────────┴────────┴─────────┴─────────┘

[a] 추가   [e] 수정   [d] 삭제   [w] 감시   [r] 새로고침   [s] 설정   [q] 종료
```

### 메뉴

| 키  | 동작     | 흐름                                                         |
| --- | -------- | ------------------------------------------------------------ |
| `a` | 추가     | 종목명 입력 → 추천 후보에서 번호 선택 → 등록                  |
| `e` | 수정     | 표에서 번호 선택 → 새 종목명 검색 → 교체                      |
| `d` | 삭제     | 표에서 번호 선택 → 삭제                                       |
| `w` | 감시     | 실시간 자동 갱신 화면 + 관련 뉴스 하단 표시. **Ctrl+C** 로 메뉴 복귀 |
| `r` | 새로고침 | 전체 시세 다시 불러오기                                      |
| `s` | 설정     | 감시 갱신 주기(초) 변경                                       |
| `q` | 종료     | 앱 종료                                                      |

### 설정

감시 모드의 갱신 주기는 `s` 설정 메뉴에서 바꿀 수 있고, `data/settings.json`
에 저장되어 다음 실행에도 유지됩니다(허용 범위 1 ~ 3600초, 기본 5초).

```json
{
  "watch_interval": 5
}
```

> 검색은 KRX에 등록된 종목명을 기준으로 합니다. 일부 종목은 영문명으로
> 등록되어 있어(예: `NAVER`) 한글로는 검색되지 않을 수 있습니다.

```text
stock_viewer_nerdy/

├── README.md
```

## 오류 처리

- 네트워크/데이터 실패 → `[ERROR] 종목 정보를 불러오지 못했습니다.`
- 검색 결과 없음 → `[INFO] '...' 와(과) 일치하는 종목이 없습니다.`
- 중복 추가 → `[INFO] 이미 등록된 종목입니다.`
- 빈 관심종목 → `[INFO] 관심 종목이 비어 있습니다.`

## 단일 EXE 빌드

```cmd
pip install pyinstaller
pyinstaller --onefile --name stock app\main.py
```

`dist\stock.exe` 가 생성됩니다. 관심종목이 유지되도록 EXE 옆에 `data` 폴더를
두세요(최초 저장 시 자동 생성).

# 와이어프레임 (Wireframe)

> 제품: **주식 관심종목 (Stock Watch)** — 터미널 TUI
> 최종 수정일: 2026-05-29 (뉴스 섹션 추가)

터미널 기반 앱이므로 화면 단위(state)별 텍스트 와이어프레임으로 표현한다.
실제 화면은 `rich` 로 컬러/박스 렌더링된다.

---

## 화면 전이도 (Screen Flow)

```
                 ┌──────────────────────┐
                 │   메인 화면 (Menu)    │◀──────────────┐
                 └──────────┬───────────┘               │
        a │ e │ d │ w │ r │ s │ q                        │
   ┌──────┼───┼───┼───┼───┼───┼──────┐                   │
   ▼      ▼   ▼   ▼   ▼   ▼   ▼      ▼                   │
 [추가] [수정][삭제][감시][새로고침][설정][종료]          │
   │      │   │     │                  │                 │
   ▼      ▼   ▼     ▼                  ▼                 │
 검색→선택  번호선택  실시간루프      주기입력             │
   │      │   │     │(Ctrl+C)         │                 │
   └──────┴───┴─────┴─────────────────┴─────────────────┘
                  (각 동작 후 메인 화면 복귀)
```

## W-1. 메인 화면 (관심종목 있음)

```
╭───────────────╮
│ 주식 관심종목 │
╰───────────────╯
                          관심 종목
┏━━━┳━━━━━━━━━━┳━━━━━━━━┳━━━━━━━━━┳━━━━━━━━┳━━━━━━━━━┳━━━━━━━━━┓
┃ # ┃ 종목명   ┃ 코드   ┃  현재가 ┃ 등락률 ┃    고가 ┃    저가 ┃
┡━━━╇━━━━━━━━━━╇━━━━━━━━╇━━━━━━━━━╇━━━━━━━━╇━━━━━━━━━╇━━━━━━━━━┩
│ 1 │ 삼성전자 │ 005930 │ 317,000 │ +5.84% │ 319,000 │ 305,500 │   ← 등락률 빨강(상승)
│ 2 │ 카카오   │ 035720 │  41,950 │ -1.20% │  44,000 │  40,450 │   ← 등락률 파랑(하락)
└───┴──────────┴────────┴─────────┴────────┴─────────┴─────────┘

[a] 추가   [e] 수정   [d] 삭제   [w] 감시   [r] 새로고침   [s] 설정   [q] 종료
선택: _
```

레이아웃 규칙
- 타이틀 패널(상단) → 관심종목 표(중앙) → 메뉴(하단) → 입력 프롬프트
- **종목명을 첫 데이터 컬럼**으로, 코드는 흐림(dim) 보조 컬럼
- 숫자 컬럼 우측 정렬, 원화 천단위 콤마

## W-4. 감시 모드 (실시간, 뉴스 포함)

```
╭─────────────────────────────╮
│ 주식 관심종목 (실시간 감시) │
╰─────────────────────────────╯
┏━━━┳━━━━━━━━━━┳━━━━━━━━┳━━━━━━━━━┳━━━━━━━━┳━━━━━━━━━┳━━━━━━━━━┓
┃ # ┃ 종목명   ┃ 코드   ┃  현재가 ┃ 등락률 ┃    고가 ┃    저가 ┃
┡━━━╇━━━━━━━━━━╇━━━━━━━━╇━━━━━━━━━╇━━━━━━━━╇━━━━━━━━━╇━━━━━━━━━┩
│ 1 │ 삼성전자 │ 005930 │ 317,000 │ +5.84% │ 319,000 │ 305,500 │
│ 2 │ 카카오   │ 035720 │  41,950 │ -1.20% │  44,000 │  40,450 │
└───┴──────────┴────────┴─────────┴────────┴─────────┴─────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 관련 뉴스 ━━━━

  1  삼성전자, 2분기 영업이익 10조 전망 상향                   Naver Finance  4분 전
     https://finance.naver.com/news/...

  2  Samsung Electronics eyes AI chip expansion amid demand surge  Yahoo Finance  12분 전
     https://finance.yahoo.com/news/...

  3  Kakao Faces Regulatory Scrutiny Over Platform Dominance      Bloomberg       1시간 전
     https://www.bloomberg.com/news/...

  4  삼성전자 HBM3E, 엔비디아 최종 승인 임박                   Naver Finance  2시간 전
     https://finance.naver.com/news/...

  5  Kakao shares rise on strong Q2 earnings beat                 Yahoo Finance  3시간 전
     https://finance.yahoo.com/news/...

5초마다 갱신 | 뉴스 3분 후 갱신 | Ctrl+C 로 메뉴 복귀
```

레이아웃 규칙
- 시세 표 아래 `━━━ 관련 뉴스 ━━━` 구분선으로 섹션 분리
- 뉴스 항목: 번호 + 제목(좌측, bold) + 출처(우측, dim) + 발행 경과 시간(dim)
  다음 줄 들여쓰기로 URL 표시 (dim, 줄 생략 없이 전체 표시)
- 최대 5건, 이후는 표시하지 않음
- 뉴스가 없으면 구분선과 섹션 전체 생략
- 뉴스 로딩 중(최초 진입 직후)이면 `[뉴스 로딩 중...]` 한 줄 표시
- 하단 안내: `{N}초마다 갱신 | 뉴스 {M}분 후 갱신 | Ctrl+C 로 메뉴 복귀`
  (뉴스 갱신까지 남은 분이 0이면 `뉴스 갱신 중...` 으로 표시)

- N초마다 화면 전체 clear 후 재렌더링 (N = 시세 갱신 설정값)
- 뉴스는 300초마다 백그라운드 파싱 후 다음 렌더링에 반영
- Ctrl+C → 백그라운드 파싱 스레드 종료 후 메인 화면 복귀

## W-5. 설정 — 다중 항목 선택

```
설정

  [1] 시세 갱신 주기       5초  (1 ~ 3600초)
  [2] 뉴스 갱신 주기     300초  (60 ~ 3600초)
  [3] 최대 뉴스 개수       5건  (1 ~ 10건)

  [0] 취소

변경할 항목 선택 (0): _
```

항목 선택 후 (예: `2` 선택 시):
```
현재값: 300초  (빈 값이면 변경 안 함)
새 뉴스 갱신 주기(초): 600

[INFO] 뉴스 갱신 주기을(를) 600초로 저장했습니다.
계속하려면 Enter: _
```

범위를 벗어난 경우:
```
[INFO] 허용 범위로 조정되어 3600초로 저장했습니다.
```

레이아웃 규칙
- 항목마다 현재값을 오른쪽 정렬(5자리)로 표시하여 한눈에 비교 가능
- 빈 입력 또는 `0` 선택 시 변경 없이 메뉴 복귀

## W-6. 수정 / 삭제 — 번호 선택

```
(관심종목 표 표시)
0) 취소

수정할 종목 번호 선택 (0): _      ← 수정
삭제할 종목 번호 선택 (0): _      ← 삭제
```

수정은 번호 선택 후 W-3과 동일한 검색→선택 흐름으로 새 종목을 고른다.

## 컬러/표기 가이드

| 요소 | 표기 |
| ---- | ---- |
| 타이틀/강조 | cyan, bold |
| 상승 등락률 | bold red |
| 하락 등락률 | bold blue |
| 보합 / 데이터 없음 | dim (`N/A`) |
| 코드 컬럼 | dim |
| 정보 메시지 | `[INFO] ...` (green/yellow) |
| 오류 메시지 | `[ERROR] ...` (red) |
| 뉴스 구분선 | dim |
| 뉴스 제목 | bold white |
| 뉴스 출처 | dim cyan |
| 뉴스 발행 시간 | dim |
| 뉴스 URL | dim (줄 바꿈 없이 전체 표시) |
| 뉴스 로딩 중 | dim italic |

# 와이어프레임 (Wireframe)

> 제품: **주식 관심종목 (Stock Watch)** — 터미널 TUI
> 최종 수정일: 2026-05-29 (뉴스 섹션 추가)

터미널 기반 앱이므로 화면 단위(state)별 텍스트 와이어프레임으로 표현한다.
실제 화면은 `rich` 로 컬러/박스 렌더링된다.

---

## 화면 전이도 (Screen Flow)

```
                 ┌──────────────────────┐
                 │   메인 화면 (Menu)    │◀──────────────┐
                 └──────────┬───────────┘               │
        a │ e │ d │ w │ r │ s │ q                        │
   ┌──────┼───┼───┼───┼───┼───┼──────┐                   │
   ▼      ▼   ▼   ▼   ▼   ▼   ▼      ▼                   │
 [추가] [수정][삭제][감시][새로고침][설정][종료]          │
   │      │   │     │                  │                 │
   ▼      ▼   ▼     ▼                  ▼                 │
 검색→선택  번호선택  실시간루프      주기입력             │
   │      │   │     │(Ctrl+C)         │                 │
   └──────┴───┴─────┴─────────────────┴─────────────────┘
                  (각 동작 후 메인 화면 복귀)
```

## W-1. 메인 화면 (관심종목 있음)

```
╭───────────────╮
│ 주식 관심종목 │   ← 주황 테두리·텍스트 (Bloomberg 주황 rgb(255,140,0))
╰───────────────╯
                           WATCHLIST
┏━━━┳━━━━━━━━━━┳━━━━━━━━┳━━━━━━━━━┳━━━━━━━━┳━━━━━━━━━┳━━━━━━━━━┓
┃ # ┃ NAME     ┃ CODE   ┃   PRICE ┃   CHG% ┃    HIGH ┃     LOW ┃   ← 앰버 헤더
┡━━━╇━━━━━━━━━━╇━━━━━━━━╇━━━━━━━━━╇━━━━━━━━╇━━━━━━━━━╇━━━━━━━━━┩
│ 1 │ 삼성전자 │ 005930 │ 317,000 │ +5.84% │ 319,000 │ 305,500 │   ← CHG%: 빨강(상승)
│ 2 │ 카카오   │ 035720 │  41,950 │ -1.20% │  44,000 │  40,450 │   ← CHG%: 파랑(하락)
└───┴──────────┴────────┴─────────┴────────┴─────────┴─────────┘

[A] 추가   [E] 수정   [D] 삭제   [W] 감시   [R] 새로고침   [S] 설정   [Q] 종료
선택: _
```

레이아웃 규칙
- 타이틀 패널(상단, 주황 테두리) → WATCHLIST 표(중앙, 앰버 헤더) → 메뉴(하단) → 입력 프롬프트
- **NAME을 첫 데이터 컬럼**으로, CODE는 밝은 회색(rgb(170,170,170)) 보조 컬럼
- 숫자 컬럼 우측 정렬, 원화 천단위 콤마
- 메뉴 키(`[A]`, `[E]` 등)는 Bloomberg 주황, 설명 텍스트는 dim

## W-4. 감시 모드 (실시간, 뉴스 포함)

```
╭──────────────────────────────╮
│ 주식 관심종목  WATCH MODE    │   ← 주황 테두리, WATCH MODE는 dim
╰──────────────────────────────╯
                           WATCHLIST
┏━━━┳━━━━━━━━━━┳━━━━━━━━┳━━━━━━━━━┳━━━━━━━━┳━━━━━━━━━┳━━━━━━━━━┓
┃ # ┃ NAME     ┃ CODE   ┃   PRICE ┃   CHG% ┃    HIGH ┃     LOW ┃
┡━━━╇━━━━━━━━━━╇━━━━━━━━╇━━━━━━━━━╇━━━━━━━━╇━━━━━━━━━╇━━━━━━━━━┩
│ 1 │ 삼성전자 │ 005930 │ 317,000 │ +5.84% │ 319,000 │ 305,500 │
│ 2 │ 카카오   │ 035720 │  41,950 │ -1.20% │  44,000 │  40,450 │
└───┴──────────┴────────┴─────────┴────────┴─────────┴─────────┘

──  NEWS  ────────────────────────────────────────────────────────   ← 주황 레이블

  1  삼성전자, 2분기 영업이익 10조 전망 상향          Naver Finance  4분 전
     https://finance.naver.com/news/...

  2  Samsung Electronics eyes AI chip expansion        Yahoo Finance  12분 전
     https://finance.yahoo.com/news/...

  3  Kakao Faces Regulatory Scrutiny                   Bloomberg       1시간 전
     https://www.bloomberg.com/news/...

  4  삼성전자 HBM3E, 엔비디아 최종 승인 임박          Naver Finance  2시간 전
     https://finance.naver.com/news/...

  5  Kakao shares rise on strong Q2 earnings beat                 Yahoo Finance  3시간 전
     https://finance.yahoo.com/news/...

────────────────────────────────────────────────────────────────
5s REFRESH  |  NEWS IN 03:00  |  Ctrl+C → MENU
```

레이아웃 규칙
- 시세 표 아래 `──  NEWS  ──` 구분선으로 섹션 분리 (주황 NEWS 레이블, 어두운 회색 선)
- 뉴스 항목: 번호(회색) + 제목(bold white) + 출처(앰버) + 발행 경과 시간(dim)
  다음 줄 들여쓰기로 URL 표시 (어두운 회색, 줄 생략 없이 전체 표시)
- 최대 5건, 이후는 표시하지 않음
- 뉴스가 없으면 `No news available.` 한 줄 표시 (dim)
- 뉴스 로딩 중(최초 진입 직후)이면 `뉴스 로딩 중...` 한 줄 표시 (dim italic)
- 하단 안내: Rule 구분선 후 `{N}s REFRESH  |  NEWS IN {MM}:{SS}  |  Ctrl+C → MENU`
  (뉴스 갱신 중이면 `NEWS UPDATING...`, 로딩 중이면 `NEWS LOADING...`)

- N초마다 화면 전체 clear 후 재렌더링 (N = 시세 갱신 설정값)
- 뉴스는 300초마다 백그라운드 파싱 후 다음 렌더링에 반영
- Ctrl+C → 백그라운드 파싱 스레드 종료 후 메인 화면 복귀

## W-5. 설정 — 다중 항목 선택

```
SETTINGS

  [1] REFRESH INTERVAL       5s   (1 ~ 3600s)
  [2] NEWS INTERVAL        300s   (60 ~ 3600s)
  [3] MAX NEWS COUNT          5   (1 ~ 10 items)

  [0] CANCEL

Select: _
```

항목 선택 후 (예: `2` 선택 시):
```
Current: 300s  (leave blank to cancel)
New NEWS INTERVAL: 600

[INFO] NEWS INTERVAL set to 600s.
계속하려면 Enter: _
```

범위를 벗어난 경우:
```
[INFO] Out of range — clamped to 3600s.
```

레이아웃 규칙
- 항목마다 현재값을 오른쪽 정렬(5자리)로 표시하여 한눈에 비교 가능
- 빈 입력 또는 `0` 선택 시 변경 없이 메뉴 복귀

## W-6. 수정 / 삭제 — 번호 선택

```
(관심종목 표 표시)
0) 취소

수정할 종목 번호 선택 (0): _      ← 수정
삭제할 종목 번호 선택 (0): _      ← 삭제
```

수정은 번호 선택 후 W-3과 동일한 검색→선택 흐름으로 새 종목을 고른다.

## 컬러/표기 가이드 (Bloomberg Terminal Style)

> 상세 명세는 [`06_style_guide.md`](./06_style_guide.md) 참조.

| 요소 | Rich 스타일 | 비고 |
| ---- | ----------- | ---- |
| 타이틀 패널 테두리·텍스트 | `bold rgb(255,140,0)` | Bloomberg 주황 |
| 테이블 제목 (WATCHLIST) | `bold rgb(255,140,0)` | Bloomberg 주황 |
| 테이블 컬럼 헤더 | `bold rgb(255,165,0)` | Bloomberg 앰버 |
| 테이블 테두리 | `rgb(80,80,80)` | 어두운 회색 |
| NAME 컬럼 | `bold white` | 핵심 식별자 |
| CODE 컬럼 | `rgb(170,170,170)` | 밝은 회색 |
| PRICE / HIGH / LOW | `white` | 일반 데이터 |
| 상승 CHG% | `bold red` | 한국 시장 관례 유지 |
| 하락 CHG% | `bold rgb(68,153,255)` | 한국 시장 관례 유지 |
| 보합 / 데이터 없음 | `dim` (`N/A`) | |
| INFO 메시지 | `rgb(255,255,0)` | 노랑 |
| ERROR 메시지 | `bold red` | |
| NEWS 구분선 (선) | `rgb(80,80,80)` | 어두운 회색 |
| NEWS 구분선 (레이블) | `bold rgb(255,140,0)` | Bloomberg 주황 |
| 뉴스 번호 | `rgb(100,100,100)` | 최소 존재감 |
| 뉴스 제목 | `bold white` | |
| 뉴스 출처 | `rgb(255,165,0)` | Bloomberg 앰버 |
| 뉴스 발행 시간 | `dim` | |
| 뉴스 URL | `rgb(100,100,100)` | |
| 메뉴 키 (`[A]`, `[E]` 등) | `bold rgb(255,140,0)` | Bloomberg 주황 |
| 메뉴 설명 텍스트 | `dim` | |
| 상태바 강조 텍스트 | `rgb(255,165,0)` | Bloomberg 앰버 |

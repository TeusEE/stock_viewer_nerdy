# 와이어프레임 (Wireframe)

> 제품: **주식 관심종목 (Stock Watch)** — 터미널 TUI
> 최종 수정일: 2026-05-29

터미널 기반 앱이므로 화면 단위(state)별 텍스트 와이어프레임으로 표현한다.
실제 화면은 `rich` 로 컬러/박스 렌더링된다.

---

## 화면 전이도 (Screen Flow)

```
                 ┌──────────────────────┐
                 │   메인 화면 (Menu)    │◀──────────────┐
                 └──────────┬───────────┘               │
        a │ e │ d │ w │ r │ s │ q                        │
   ┌──────┼───┼───┼───┼───┼───┼──────┐                   │
   ▼      ▼   ▼   ▼   ▼   ▼   ▼      ▼                   │
 [추가] [수정][삭제][감시][새로고침][설정][종료]          │
   │      │   │     │                  │                 │
   ▼      ▼   ▼     ▼                  ▼                 │
 검색→선택  번호선택  실시간루프      주기입력             │
   │      │   │     │(Ctrl+C)         │                 │
   └──────┴───┴─────┴─────────────────┴─────────────────┘
                  (각 동작 후 메인 화면 복귀)
```

---

## W-1. 메인 화면 (관심종목 있음)

```
╭───────────────╮
│ 주식 관심종목 │
╰───────────────╯
                          관심 종목
┏━━━┳━━━━━━━━━━┳━━━━━━━━┳━━━━━━━━━┳━━━━━━━━┳━━━━━━━━━┳━━━━━━━━━┓
┃ # ┃ 종목명   ┃ 코드   ┃  현재가 ┃ 등락률 ┃    고가 ┃    저가 ┃
┡━━━╇━━━━━━━━━━╇━━━━━━━━╇━━━━━━━━━╇━━━━━━━━╇━━━━━━━━━╇━━━━━━━━━┩
│ 1 │ 삼성전자 │ 005930 │ 317,000 │ +5.84% │ 319,000 │ 305,500 │   ← 등락률 빨강(상승)
│ 2 │ 카카오   │ 035720 │  41,950 │ -1.20% │  44,000 │  40,450 │   ← 등락률 파랑(하락)
└───┴──────────┴────────┴─────────┴────────┴─────────┴─────────┘

[a] 추가   [e] 수정   [d] 삭제   [w] 감시   [r] 새로고침   [s] 설정   [q] 종료
선택: _
```

레이아웃 규칙
- 타이틀 패널(상단) → 관심종목 표(중앙) → 메뉴(하단) → 입력 프롬프트
- **종목명을 첫 데이터 컬럼**으로, 코드는 흐림(dim) 보조 컬럼
- 숫자 컬럼 우측 정렬, 원화 천단위 콤마

## W-3. 추가 — 검색 후보 선택

```
추가할 종목명을 입력하세요 (예: 삼성): 삼성

┏━━━┳━━━━━━━━━━━━┳━━━━━━━━┳━━━━━━━┓
┃ # ┃ 종목명     ┃ 코드   ┃ 시장  ┃
┡━━━╇━━━━━━━━━━━━╇━━━━━━━━╇━━━━━━━┩
│ 1 │ 삼성전자   │ 005930 │ KOSPI │
│ 2 │ 삼성전자우 │ 005935 │ KOSPI │
│ 3 │ 삼성SDI    │ 006400 │ KOSPI │
└───┴────────────┴────────┴───────┘
0) 취소

번호 선택 (0): _
```

선택 후:
```
[INFO] 삼성전자 추가 완료.
계속하려면 Enter: _
```

## W-4. 감시 모드 (실시간)

```
╭─────────────────────────────╮
│ 주식 관심종목 (실시간 감시) │
╰─────────────────────────────╯
┏━━━┳━━━━━━━━━━┳━━━━━━━━┳━━━━━━━━━┳━━━━━━━━┳━━━━━━━━━┳━━━━━━━━━┓
┃ # ┃ 종목명   ┃ 코드   ┃  현재가 ┃ 등락률 ┃    고가 ┃    저가 ┃
┡━━━╇━━━━━━━━━━╇━━━━━━━━╇━━━━━━━━━╇━━━━━━━━╇━━━━━━━━━╇━━━━━━━━━┩
│ 1 │ 삼성전자 │ 005930 │ 317,000 │ +5.84% │ 319,000 │ 305,500 │
│ 2 │ 카카오   │ 035720 │  41,950 │ -1.20% │  44,000 │  40,450 │
└───┴──────────┴────────┴─────────┴────────┴─────────┴─────────┘

5초마다 갱신됩니다. 메뉴로 돌아가려면 Ctrl+C 를 누르세요.
```

- N초마다 화면 전체 clear 후 재렌더링 (N = 설정값)
- Ctrl+C → 메인 화면 복귀

---

## W-5. 설정 — 갱신 주기 변경

```
설정  현재 감시 갱신 주기: 5초
(1 ~ 3600초, 빈 값이면 변경 안 함)
새 갱신 주기(초): 10

[INFO] 감시 갱신 주기를 10초로 저장했습니다.
계속하려면 Enter: _
```

범위를 벗어난 경우:
```
[INFO] 허용 범위로 조정되어 3600초로 저장했습니다.
```

---

## W-6. 수정 / 삭제 — 번호 선택

```
(관심종목 표 표시)
0) 취소

수정할 종목 번호 선택 (0): _      ← 수정
삭제할 종목 번호 선택 (0): _      ← 삭제
```

수정은 번호 선택 후 W-3과 동일한 검색→선택 흐름으로 새 종목을 고른다.

## 컬러/표기 가이드

| 요소 | 표기 |
| ---- | ---- |
| 타이틀/강조 | cyan, bold |
| 상승 등락률 | bold red |
| 하락 등락률 | bold blue |
| 보합 / 데이터 없음 | dim (`N/A`) |
| 코드 컬럼 | dim |
| 정보 메시지 | `[INFO] ...` (green/yellow) |
| 오류 메시지 | `[ERROR] ...` (red) |
